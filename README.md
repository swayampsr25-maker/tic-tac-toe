Tic‑Tac‑Toe (by Swayam Prakash Swain)
Live Demo: https://swayam-25.github.io/Tic-Tac-Toe/

Overview
This is a simple, fun 3×3 grid Tic‑Tac‑Toe game built using HTML, CSS and JavaScript. It was developed by Swayam Prakash Swain as a browser‑playable game to practise front‑end coding.

Features
- Two‑player gameplay (Player O and Player X)
- Score tracking for both players
- Start / Cancel / End / Reset / Continue controls
- Responsive and clean UI
- Smooth win/draw detection

Technologies Used
- HTML – Structure
- CSS – Styling
- JavaScript – Game logic

Project Structure
/tic‑tac‑toe
│── index.html      → UI layout  
│── style.css       → Styling  
│── script.js       → Game logic  
└── assets/         → Optional images/icons  

How to Run
1. Download/clone the project.  
2. Open index.html in any browser.  
3. Choose your symbol (O or X).  
4. Play by clicking the cells.  
5. Use Reset/Continue as needed.

How the Game Works
1. Player selects a symbol.  
2. Players take turns marking cells.  
3. JavaScript checks for:
   - Horizontal win
   - Vertical win
   - Diagonal win
   - Draw  
4. Score updates and game can be continued or ended.
